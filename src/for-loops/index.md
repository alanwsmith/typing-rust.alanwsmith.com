# For Loops

For loops allow us to run the same block of
code multiple times.

Here a counter that goes up from `1` to `5`.

#### SOURCE CODE

```rust, noplayground, EXAMPLE1
fn main() {

  for alfa in (1..=5) {
    println!("alfa is {}", alfa);
  }

}
```
