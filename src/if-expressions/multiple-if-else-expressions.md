# Multiple if else Expressions

#### SOURCE CODE

```rust, noplayground, EXAMPLE1
fn main() {

  let value = 7;

  if value <= 5 {
    println!("alfa");

  } else if value <= 6 {
    println!("bravo");

  } else if value <= 7 {
    println!("charlie");

  } else if value <= 8 {
    println!("delta");

  } else {
    println!("echo");
  }

}
```
