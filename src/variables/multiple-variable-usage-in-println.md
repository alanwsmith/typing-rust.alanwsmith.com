# Multiple Variable Usage in println!

Adding multiple placeholders and passing
the vairalbe multiple times allows us
to use it more than once in the output.

For example:

#### SOURCE CODE

```rust,noplayground,EXAMPLE1
fn main() {

  let alfa: i32 = 27;

  println!("{} - {} - {}", alfa, alfa, alfa);

}
```
